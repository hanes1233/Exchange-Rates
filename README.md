# Exchange Rates 

## Description 

Spring-boot application, returns exchange rates of some currencies(table from database or details of exact currency).

## How to use

Application works on localhost with connected database(PostgreSQL), runs on maven.
Starts on mainpage(HTML), user has choice of 2 queries : load whole database, or one specific elements using HTML form, which redirect him on respectivly page.
